# Grabz — Vinted Monitor & Autobuy Bot

The fastest Vinted monitor on the market — completely free. Every new listing, delivered to your Discord in under a second.

![Grabz Preview](assets/preview.gif)

## Features

- **Real-time monitoring** — covers all of Europe and the USA, no item missed
- **Smart filtering** — price, brand, size, color, keywords, regex, seller filters
- **One-click autobuy** — buy, offer, or favourite directly from Discord
- **Dashboard** — manage everything from an interactive Discord panel
- **Multi-currency** — automatic conversion across 13+ currencies
- **Rich embeds** — images, price, seller info, and action buttons on every item
